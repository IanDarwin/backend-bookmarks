DarwinSys Cloud-Native Bookmarks Application

Besides being useful, this webapp is a Quarkus cloud application demo; the source code is available from my github repo.

Quarkus is a framework for quickly building cloud-native microservices in Java Jakarta-EE.

You need to access this via a REST web service client, not a browser. To see what the service sends, you can try these examples: (output in JSON)

  • List of categories (topics)
  • List of bookmarks in category 'politics'
  • List of bookmarks with string 'so much' in description
  • List of bookmarks with regex 'so.*much' in description
  • Bookmark #1
